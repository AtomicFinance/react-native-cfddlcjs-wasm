# react-native-cfddlcjs-wasm

WebView wrapper to run cfddlcjs-wasm in React Native.

## Installation

```sh
npm install react-native-cfddlcjs-wasm
```

## Usage

```js
import { initCfddlc } from "react-native-cfddlcjs-wasm";

// ...

const cfddlc = await initCfddlc();
```
